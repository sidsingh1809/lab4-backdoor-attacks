Dependencies -

Python 3.6.9
Keras 2.3.1
Numpy 1.16.3
Matplotlib 2.2.2
H5py 2.9.0
TensorFlow-gpu 1.15.2

Github - https://github.com/sidsingh1809/lab4-backdoor-attacks

Data - https://github.com/csaw-hackml/CSAW-HackML-2020/tree/master/lab3
